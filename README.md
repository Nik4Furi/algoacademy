## <h2 id="runLocally" >Run Locally </h2>

Clone the project

```bash
  git clone https://github/Nik4Furi/algoacademy
```

Go to the project directory

```bash
  cd algoacademy
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
